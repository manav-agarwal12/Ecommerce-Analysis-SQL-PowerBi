
# Dataset

This folder contains sample datasets used for the E-commerce Sales Analysis project.

Due to GitHub file size limitations, the complete dataset is not stored in this repository.

**Original Dataset Source:**
https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

The dataset includes information on customers, orders, products, sellers, payments, reviews, and geolocation data from a Brazilian e-commerce platform.

Please download the full dataset from Kaggle if you wish to reproduce the complete analysis.
